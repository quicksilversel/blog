## 2. zsh-autosuggestions

## 4. fzf

**GitHub**: [junegunn/fzf](https://github.com/junegunn/fzf)

Fuzzy finder for command history, files, and directories.

### Install

```bash
brew install fzf

# Install keybindings
$(brew --prefix)/opt/fzf/install
```

Sets up:

- `Ctrl+R` - Search command history
- `Ctrl+T` - Search files
- `Alt+C` - Search directories and cd

## 5. zoxide

**GitHub**: [ajeetdsouza/zoxide](https://github.com/ajeetdsouza/zoxide)

Smart cd replacement that learns directory patterns.

### Install

```bash
brew install zoxide

# Add to ~/.zshrc
eval "$(zoxide init zsh)"
```

### Usage

Instead of:

```bash
cd ~/Projects/work/frontend/src/components/buttons
```

Just:

```bash
z buttons
```

zoxide remembers frequently visited directories and jumps there directly.

Fuzzy matching works too:

```bash
z blog art  # â†’ ~/Projects/blog/articles
```

After a week, muscle memory kicks in:

```bash
z proj  # â†’ ~/Projects/
z comp  # â†’ ~/Projects/work/frontend/src/components/
```

## Git Aliases

Save keystrokes on git commands.

### Setup

```bash
git config --global alias.st status
git config --global alias.co checkout
git config --global alias.br branch
git config --global alias.cm 'commit -m'
git config --global alias.ca 'commit --amend'
git config --global alias.p push
git config --global alias.pl pull
git config --global alias.l 'log --oneline --graph --decorate'
git config --global alias.undo 'reset HEAD~1 --soft'
```

### Usage

Instead of:

```bash
git status
git checkout -b feature/dashboard
git commit -m "feat: add dashboard"
git push origin feature/dashboard
```

Now:

```bash
git st
git co -b feature/dashboard
git cm "feat: add dashboard"
git p origin feature/dashboard
```

Useful aliases:

**`git l`** - Pretty one-line log:

```bash
* 2a0dfc5 (HEAD -> main) fix: update API endpoint
* df8fa26 feat: add dashboard component
* 1067bbf refactor: clean up old code
```

**`git undo`** - Undo last commit, keep changes:

```bash
git cm "typo in message"
git undo  # Keeps changes staged
git cm "fix: correct message"
```

## History Configuration

Default zsh history is limited. Fix it:

```bash
# Add to ~/.zshrc
HISTFILE=~/.zsh_history
HISTSIZE=50000
SAVEHIST=50000
setopt APPEND_HISTORY
setopt SHARE_HISTORY
```

**HISTSIZE/SAVEHIST=50000** - Store 50k commands instead of 2k

**APPEND_HISTORY** - Append to history file, don't overwrite

**SHARE_HISTORY** - Share history across all terminal tabs immediately
